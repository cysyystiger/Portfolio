This lab practice is a course in EE department, requiring us to design, weld and test whether the circuit could work.

Design Process

Design Circuit

First, we design circuit by drawing on the SIMPLIS.

The functions of the circuit is annotated by TA, Teng-Wei Tsai.

Simulation

Make sure that the circuit would perform functions we expect. We continue to run the simulation on SIMPLIS. The following figures just show a couple of features we had examined.

Draw Layout

This part is the most annoying one. We have to draw the circuit like in the SIMPLIS, besides, annotating positive or negative end on the figure is necessary, which makes us more convenient to arrange wires in the layout by Altium Designer.

The thickness of the wire is very crucial. Through the simulation, we know the current value of every wire. The larger the current, the thicker the wire. That's because we want to minimize aberrations from the theoretical values. The space for layout is limited, so we have to take lots of time to put elements and wires neatly.

Engrave

We put the layout file into machine, and weld all elements on the board carefully.

Test

After finishing the circuit, we proceed with the last step. We test the circuit by examining whether the waves shown by the oscilloscope is anomalous. Still, I just show some of the figures we had examined.

Besides these tests, we also run the burn-in test. The temperature is steady at 65 degrees Celsius for thirty minutes.

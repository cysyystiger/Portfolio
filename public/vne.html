Introduction

Imagine there is a huge substrate network, and you want to put many little virtual networks in the substrate network. Notice that the substrate network is composed of some nodes and links, and every node and link would be given a value referred to as the capability of CPU and the bandwidth between two computers. The following figure is examples of a substrate and a virtual network.

Now, we go to the problem formulation, how do we put virtual networks as many as possible? Of course, we should follow the limited resource of the substrate network. This problem has proven to be an NP-complete problem. Therefore, we try to simulate methods provided by papers to examine which algorithm is better.
